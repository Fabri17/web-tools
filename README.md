## 🎯 ¿Qué es WebTools Suite?

Suite completa de **herramientas de desarrollo** que funcionan 100% en tu navegador. Sin instalación, sin registro, sin rastreadores.

### Por qué usar WebTools Suite

- ✅ **100% Local**: Todo se procesa en tu navegador
- ✅ **Privacidad Total**: Cero datos enviados a servidores
- ✅ **Gratis para Siempre**: Sin planes premium ni límites artificiales
- ✅ **Sin Complicaciones**: Abrir y usar, sin configuración
- ✅ **Código Abierto**: Auditable y transparente

## 🚀 Inicio Rápido

### Desarrollo Local

```bash
# Instalar dependencias
pnpm install

# Iniciar servidor de desarrollo
pnpm dev

# Build para producción
pnpm build

# Preview del build
pnpm preview
```

### Build y Empaquetado

```bash
# Linux/macOS
bash build-and-package.sh

# Windows PowerShell
powershell -ExecutionPolicy Bypass -File build-and-package.ps1
```

Genera un archivo ZIP listo para subir a **Netlify**, **Vercel**, **GitHub Pages**, etc.

## 🔒 Privacidad y Seguridad

### Compromiso de Privacidad

- ✅ **100% procesamiento local**: Ningún archivo sube a servidores
- ✅ **Sin rastreadores de terceros**: Solo analytics anónimo opcional
- ✅ **Sin cookies de seguimiento**: Solo esenciales para funcionalidad
- ✅ **Código abierto**: Audita tú mismo el código
- ✅ **Sin cuentas requeridas**: Sin registro, sin login

### Seguridad

- CSP (Content Security Policy) estricto
- Headers de seguridad configurados
- Sanitización de inputs
- No ejecuta código del usuario en servidor
